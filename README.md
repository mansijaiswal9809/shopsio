# This is an RESPONSIVE E-commerce Application with following functionalities
* Filter
* Search
* Add to cart
* Login , Logout
## Techs & Tools Used
* For frontend: HTML5, CSS3, ReactJS, JavaScript
* CSS Library: Tailwind CSS
* For Authentication: Auth0
## Hooks Used
* useState()
* useEffect()
* useReducer()
* useContext()
